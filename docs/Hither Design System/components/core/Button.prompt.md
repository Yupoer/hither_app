Chunky pill-shaped primary action button; springy scale-down on press, accent glow on hover. Use for the main call-to-action on any screen.

```jsx
<Button variant="primary" full iconLeft="🚩">設定集合點</Button>
<Button variant="glass" size="sm">稍後</Button>
```

Variants: `primary` (Signal Orange), `secondary` (Electric Sky), `success` (Grass green), `ghost` (charcoal), `glass` (Liquid Glass). Sizes: `lg` (54px), `sm` (40px).
