# ios_native/ — archived

The native Swift projects that previously lived here (`hither/` and `hitherUI/`) have been deleted.

Key patterns and native capability implementations were extracted into:

**`../docs/legacy-ios-reference.md`** — design system, domain models, UWB precision finding, Live Activity, battery-aware location, and performance patterns.

The active app is **React Native** (`apps/mobile`) with thin native modules under `apps/mobile/modules/` and Supabase as the backend.
