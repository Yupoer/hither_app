# hitherUI/ — FOR REFERENCE ONLY

This is the standalone **SwiftUI design prototype** (LoginView, DashboardView,
MapView, ItineraryView, GroupSetupView, SettingsView, plus `draftUI_reference/`
mockups). It was a UI/UX exploration and is **not part of the shipping app**.

The real, cross-platform UI is **React Native** in `apps/mobile/src/screens`
(Route A). Use these SwiftUI screens and the `draftUI_reference/` images only
as a visual/behaviour reference — do not build on or ship them.

Left in place (not deleted) so the design intent stays available. Safe to
remove once the RN screens fully cover these flows.
