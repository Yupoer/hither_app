## OpenAI Codex and GPT-5.6 Usage

This project was built with assistance from OpenAI Codex and GPT-5.6.

- **Architecture and planning:** We used GPT-5.6 to review the product scope, define the Leader–Follower group coordination model, and structure the React Native, Expo, and Supabase architecture.
- **Implementation:** We used Codex to accelerate implementation across React Native screens, Expo platform configuration, Supabase schema and migration work, location-service abstractions, and test scaffolding.
- **Debugging and validation:** We used Codex and GPT-5.6 to investigate type errors, review edge cases in group membership and location updates, and improve test coverage.
- **Product iteration:** We used GPT-5.6 to refine onboarding flows, feature scope, and UX copy for group creation, joining, meeting points, and straggler alerts.

All final implementation decisions, code integration, testing, and deployment configuration were reviewed and completed by the team.
